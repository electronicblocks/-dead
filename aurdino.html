Blinking led on Arduino

Default led is given at pin 13.

Source code of this project

// program to blink led
int ledpin=13;
void setup() {
pinMode(ledpin,OUTPUT);
}
void loop() {
digitalWrite(ledpin,HIGH);
delay(500);
digitalWrite(ledpin,LOW);
delay(500);
}.
